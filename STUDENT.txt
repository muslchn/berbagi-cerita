MAIN_FEATURES:
- Register, login, logout, and protected routes
- Story list from Dicoding Story API
- Search and sort on the story list
- Interactive map with story markers and multiple map layers
- Add story form with description, photo upload, camera capture, and map location picker
- Push notification subscribe and unsubscribe from navigation and About page
- PWA installability with service worker and web app manifest
- Offline app shell support
- User-selected saved stories in IndexedDB
- Saved stories page with delete action
- Pending offline story queue for later sync

INDEXEDDB_STORES:
- savedStories: stories saved by explicit user action
- pendingStories: stories waiting to be submitted when the app is online again

REVIEWER_NOTES:
- Register a new account from #/register if no prepared account is available.
- Login is required before opening #/, #/map, #/add, and #/saved.
- To test saved stories, click "Simpan Cerita" on the home page, then open #/saved.
- To test push notification, login, enable notification from navigation or #/about, then add a new story.
- To test offline behavior, open the app once, save at least one story, then enable offline mode from DevTools.
